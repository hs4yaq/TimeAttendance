1. การเริ่มต้นใช้งาน

การติดตั้งแอป (PWA)

เพื่อความรวดเร็วและให้ระบบ GPS ทำงานได้เสถียรที่สุด แนะนำให้ติดตั้งระบบไว้บนหน้าจอโฮม

  • Android: กดปุ่ม "ติดตั้ง" ที่เด้งขึ้นมาด้านล่างจอ
  • iOS: เปิดด้วยแอป Safari > กดปุ่มแชร์ (Share) > เลือก "เพิ่มไปยังหน้าจอโฮม"
ระบบไม่อนุญาตให้เปิดผ่านแอป LINE หรือ Facebook เพื่อป้องกันพิกัดเพี้ยน

การลงทะเบียนครั้งแรก

บุคลากรใหม่ต้องทำการลงทะเบียนเครื่องโทรศัพท์ของตนเอง (1 บัญชี ต่อ 1 เครื่องเท่านั้น)

  1. กดปุ่ม "เข้าใช้งานครั้งแรกที่นี่" จากหน้าล็อกอิน
  2. เลือกหมวดหมู่: ผู้บริหาร / ครู / เจ้าหน้าที่
  3. กรอกข้อมูลให้ครบถ้วน และระบุ "รหัสประจำตัว"
  4. กดบันทึกเพื่อผูกเครื่องโทรศัพท์เครื่องนี้เข้ากับระบบทันที

การล็อกอินครั้งถัดไป: เพียงกรอก "รหัสผู้ใช้งาน" ในหน้าหลัก ระบบจะพาเข้าสู่หน้าลงเวลาทันที

3. สำหรับผู้ดูแลระบบ (Admin)

การเข้าถึงสิทธิ์ Admin

ระบบป้องกันความปลอดภัยโดยใช้ Google Login ผู้ที่จะเข้าหน้าแอดมินได้ ต้องใช้อีเมล (`@ptc.ac.th` หรือ `@gmail.com`) ที่ถูกเพิ่มชื่อไว้ในระบบหลังบ้านเท่านั้น หากอีเมลไม่ตรง ระบบจะดีดออกทันที

** สามารถเข้าทางปุ่ม "ผู้ดูแลระบบ" (หน้าล็อกอิน) หรือไอคอนรูปกุญแจในหน้า Dashboard

แผงควบคุมหลัก (Admin Panel)

แบ่งออกเป็น 2 แท็บหลักเพื่อให้ง่ายต่อการจัดการข้อมูลรายวัน

แท็บ: สรุปตามกลุ่มสังกัด

  • ดูภาพรวมยอดคนลงเวลาเข้า-ออกงาน (แบบ Real-time)
  • แยกข้อมูลตามหมวดหมู่: ผู้บริหาร, ฝ่ายสนับสนุน, สายวิชาการ
  • การใส่หมายเหตุ: หากพนักงานขาด/ลา แอดมินสามารถเลือกประเภทการลา (เช่น ลากิจ, ลาป่วย, ไปราชการ) ได้โดยตรงจากหน้านี้ ข้อมูลจะบันทึกลงฐานข้อมูลและแสดงในรายงานทันที

แท็บ: จัดการรายชื่อ/เครื่อง

  • ค้นหารายชื่อบุคลากรทั้งหมดในระบบอย่างรวดเร็ว
  • ปุ่ม เพิ่ม/แก้ไขข้อมูล: หากพนักงานพิมพ์ชื่อหรือแผนกผิด แอดมินแก้ให้ได้
  • ปุ่ม รีเซ็ตอุปกรณ์ (สีเหลือง): ใช้เมื่อพนักงานเปลี่ยนมือถือใหม่ กดปุ่มนี้เพื่อลบการผูกเครื่องเก่าออก พนักงานจะล็อคอินเครื่องใหม่ได้ทันที

การตั้งค่าโครงสร้างวิทยาลัย

คลิกที่ปุ่มฟันเฟืองมุมขวาบนของหน้า Admin เพื่อเปิดหน้าต่างนี้ (รองรับการตั้งค่าทุกอย่างโดยไม่ต้องแก้โค้ด)

1. ระบบ & เวลา

ตั้งค่า Lat/Lng ของวิทยาลัย, รัศมีวงกลม (ม.), และเวลาเปิด-ปิดปุ่มกดเช็คอิน/เช็คเอาท์

2. ควบคุมช่องทาง (ใหม่)

กดที่ไอคอน LINE/Facebook เพื่อบล็อกการเข้าถึงจากแอปแชท ป้องกันปัญหาพิกัดเพี้ยน

3. โครงสร้างบุคลากร

พิมพ์กำหนดรายชื่อผู้บริหาร, ฝ่ายสนับสนุน, และแผนกวิชาการ (1 บรรทัดต่อ 1 แผนก)

4. ข้อมูลอื่นๆ

กำหนดรายการอีเมลแอดมิน, รูปแบบชื่อตำแหน่ง, และประเภทการลาในระบบ

รองรับการ Export ข้อมูลประจำวันออกเป็นไฟล์ Word เพื่อนำไปจัดหน้าพิมพ์ต่อ หรือสั่งพิมพ์เป็น PDF ทันที และมีหน้า System Logs บันทึกประวัติการกระทำของแอดมินทุกขั้นตอน

4. ปัญหาที่พบบ่อย (FAQ)

กดปุ่มไม่ได้ ระบบฟ้องว่า "พิกัด GPS ล้มเหลว"
สาเหตุ: เบราว์เซอร์หรือมือถือไม่ได้รับอนุญาตให้เข้าถึงตำแหน่งที่ตั้ง (Location Services)

วิธีแก้ (สำหรับ iOS/iPhone):
1. ไปที่การตั้งค่า (Settings) เครื่อง > ความเป็นส่วนตัว > บริการหาตำแหน่งที่ตั้ง
2. เลื่อนหา "เว็บไซต์ Safari" แล้วเลือกเป็น "ในระหว่างใช้แอป" หรือ "ถามครั้งถัดไป"
3. หากทำแล้วยังไม่ได้ ให้ลบไอคอนแอปออกจากหน้าจอโฮม แล้วเข้าผ่าน Safari เพื่อกดเพิ่มลงหน้าจอใหม่อีกครั้ง
ระบบแจ้งว่า "อุปกรณ์ไม่ได้รับอนุญาต"
สาเหตุ: บุคลากรมีการเปลี่ยนเครื่องโทรศัพท์มือถือ หรือเผลอลบข้อมูลคุกกี้ในเครื่อง ทำให้ Device ID เปลี่ยนไป ระบบจึงมองว่าเป็นเครื่องอื่น

วิธีแก้:
ให้พนักงานแจ้งแอดมินระบบ แอดมินเข้าไปที่หน้า Admin Panel > แท็บที่ 2 (จัดการรายชื่อ) > ค้นหาชื่อพนักงาน > กดปุ่มรีเซ็ตสีเหลือง หลังชื่อพนักงานคนนั้น จากนั้นพนักงานจะสามารถใช้เครื่องใหม่ล็อคอินได้ทันที
หน้าจอขึ้นว่า "ช่องทางนี้ถูกปิดการใช้งาน"
สาเหตุ: แอดมินตั้งค่าไม่อนุญาตให้เปิดลิงก์ผ่านแอปนั้นๆ (เช่น LINE หรือ Facebook In-App Browser) เพื่อป้องกันปัญหาพิกัดแผนที่กระโดดหรือไม่ทำงาน

วิธีแก้:
ให้กดที่จุดสามจุด (มุมขวาบน หรือมุมขวาล่างของจอ) แล้วเลือกคำสั่ง "เปิดด้วยเบราว์เซอร์เริ่มต้น" (Open in Default Browser) หรือคัดลอกลิงก์ไปเปิดในแอป Safari / Chrome แทน
